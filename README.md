# Todo App with Gamification

This is a code bundle for a Todo application that incorporates gamification elements to make task management more engaging.

## Features

- **Task Management**: Create, track, and complete daily tasks.
- **Gamification**: Experience rewards and visual feedback (like confetti) upon task completion.
- **Interactive UI**: Drag-and-drop functionality for task organization.
- **Data Visualization**: Progress tracking using charts.
- **Responsive Design**: Optimized for various screen sizes.

## Tech Stack

- **Framework**: React
- **Build Tool**: Vite
- **Styling**: Tailwind CSS, MUI (Material UI), Radix UI
- **Animations**: Motion (Framer Motion)
- **Icons**: Lucide React
- **Charts**: Recharts
- **Drag & Drop**: React-dnd
- **Other Utilities**: React Router, Date-fns, Canvas-confetti

## Running the Code

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Start the development server**:
   ```bash
   npm run dev
   ```

3. **Build for production**:
   ```bash
   npm run build
   ```
